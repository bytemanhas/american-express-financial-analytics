# American Express Financial Analytics & Risk Intelligence Dashboard

An end-to-end Financial Analytics and Risk Intelligence project built using Python, SQL, and Power BI to analyze customer spending behavior, revenue trends, payment performance, customer segmentation, and credit risk exposure.

This project simulates a real-world financial analytics use case for American Express using synthetic banking data generated with Python and Faker.

---

# Project Objective

The goal of this project was to build an interactive analytics solution to help stakeholders monitor:

- Revenue performance
- Customer spending behavior
- Customer segmentation
- Credit risk exposure
- Payment behavior
- High-risk customers

The dashboard enables data-driven decision-making for business, operations, and risk teams.

The project uses 4 synthetic datasets generated using Python.

- Transactions Dataset → 50,000 records

# Project Workflow

## Phase 1 — Data Generation & ETL
- Generated synthetic financial datasets using Python and Faker
- Created customer, transaction, payment, and credit risk data

## Phase 2 — Python EDA
Performed exploratory analysis to understand:
- Revenue trends
- Segment contribution
- Customer spending behavior
- Payment delays
- Credit risk distribution

## Phase 3 — SQL Analysis
Performed business-focused SQL analysis for:
- Top spending customers
- Revenue by segment
- Risk-based customer analysis
- High spend high-risk customers

## Phase 4 — Power BI Dashboard
Built a premium 3-page interactive dashboard using Power BI and DAX.

# Key Business Insights

- Total Revenue generated: **$54.9M**
- Average Transaction Value: **$1,098**
- High-value customers contribute **60% of total revenue**
- Travel is the highest spending category
- 17.3% customers fall under high-risk category
- Lower-income customers show higher credit risk exposure
- Missed payments strongly correlate with risk classification
